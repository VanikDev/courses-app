## Project Courses with based on Express

🚀 Руководство по запуску проекта
Этот проект работает на Node.js с использованием Mongoose и шаблонизатора Handlebars (HBS).
📋 Предварительные требования
Перед запуском убедитесь, что у вас установлены:

- Node.js (версия 24.14.0 или выше)
- MongoDB (ссылка на Atlas)

🛠 Установка

1.  Склонируйте репозиторий или скачайте архив с кодом.
2.  Откройте терминал в папке проекта.
3.  Установите все необходимые зависимости:

```bash
npm ci
```

🏃 Запуск проекта
В проекте настроено два основных сценария запуска. Мы рекомендуем использовать Dev-режим для разработки.

1. Режим разработки (Рекомендуется)
   Использует nodemon, который автоматически перезагружает сервер при каждом сохранении изменений в коде.

```bash
npm run dev
```

2. Обычный запуск
   Стандартный запуск сервера через Node.js без автоперезагрузки.

```bash
npm start
```

---

🏗 Структура и особенности

- База данных: Проект подключается к MongoDB через Mongoose. Проверьте строку подключения в index.js.
- Шаблоны: Для рендеринга страниц используется Handlebars.
- Номера заказов: В шаблонах реализован хелпер increment, который выводит порядковый номер заказа, используя переменную {{@index}}.

Пример вывода заказа в HBS:

```hbs
<span class='card-title order-count'>
  Заказ №{{increment @index}}:
  <small>{{_id}}</small>
</span>
```
